Teztin Website

A modern, responsive website with glassmorphism design, featuring a functional contact form with database integration.

Features

  • Glassmorphism Design - Beautiful glass-like UI elements
  • Parallax Background - Dynamic bubble animations on scroll
  • Contact Form - Fully functional with validation
  • Database Integration - PostgreSQL backend for data storage
  • Responsive Design - Works on all devices
  • Real-time Validation - Form validation with error feedback

Quick Start

Prerequisites

  • Node.js (v16 or higher)
  • PostgreSQL (v13 or higher)

Installation

  1. Navigate to the project 
    cd teztin-website
  2. Install dependencies 
    npm install
  3. Set up environment 
    cp glassmorphism.env .env
# Edit .env with your database credentials
  4. Set up database 
    createdb teztin_contacts
  5. Start the backend 
    npm start
# Or for development: npm run dev
  6. Open the website 
    # Open glassmorphism.html in your browser
# Or serve with a local server
python -m http.server 8000
# Then visit: http://localhost:8000/glassmorphism.html

How It Works

Frontend (glassmorphism.html)

  • Glassmorphism UI - Modern glass-like design elements
  • Parallax Effects - Background bubbles move on scroll
  • Form Validation - Real-time input validation
  • API Integration - Sends data to backend server

Backend (glassmorphism-backend.js)

  • Express Server - RESTful API endpoints
  • PostgreSQL - Database for storing contact submissions
  • CORS Support - Cross-origin requests enabled
  • Error Handling - Comprehensive error management

Database Access

View All Contacts

# API endpoint
curl http://localhost:3001/api/contacts

# Or visit in browser
http://localhost:3001/api/contacts

Direct Database Access

# Connect to database
psql -U postgres -d teztin_contacts

# View contacts
SELECT * FROM contacts ORDER BY created_at DESC;

Configuration

Environment Variables (.env)

DB_HOST=localhost
DB_PORT=5432
DB_NAME=teztin_contacts
DB_USER=postgres
DB_PASSWORD=your_password
PORT=3001

Database Schema

CREATE TABLE contacts (
    id SERIAL PRIMARY KEY,
    name VARCHAR(255) NOT NULL,
    email VARCHAR(255) NOT NULL,
    company VARCHAR(255),
    description TEXT NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
